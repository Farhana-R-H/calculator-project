# Calculator Project

A responsive web-based calculator built with django backend and frontend using **HTML, CSS, and JavaScript**, designed to handle predefined pricing and quantity inputs with automatic total calculation. The project is mobile-friendly and desktop-ready with smooth interactive elements.

---

## Features

- **Predefined price options** with dropdown selection.
- **Quantity input** for each item with validation.
- **Tick/confirm button** that locks the selection and calculates subtotal.
- **Automatic total calculation** with a dedicated total box.
- **Responsive design** for both desktop and mobile screens.
- **Reset button** to clear all selections and start over.
- **Interactive UI** with hover effects and user-friendly dropdowns.

---

## Technologies Used

- **HTML5** – Structure of the calculator.
- **CSS3** – Styling, layout, and responsive design using flexbox and media queries.
- **JavaScript (Vanilla JS)** – Dynamic row creation, dropdown handling, tick logic, subtotal and total calculation.
